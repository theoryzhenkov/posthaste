# Design Context

Canonical design docs live in `docs/`. This file bridges them for impeccable skills.
Do not duplicate details here — read the source docs for full context.

## Target audience

Power email users (MailMate, mutt, Thunderbird power-user crowd), but welcoming to newcomers.
Open-source project; guides and onboarding are first-class priorities.

> Source: [docs/L0-branding.md](docs/L0-branding.md#audience)

## Use cases

Desktop email client — reading, triaging, archiving, and managing mail across multiple JMAP accounts.
Keyboard-first, information-dense, always-connected. No offline or mobile-first for v1.

> Source: [docs/L0-ui.md](docs/L0-ui.md)

## Brand personality / tone

"Gentle brutalist" — information-dense and serious when you need it, friendly and joyful when you don't.
A professional power tool that is also a pleasure to use.
Energy: speed, wind, lightness, sky, organic movement. Not heavy industrial.

> Source: [docs/L0-branding.md](docs/L0-branding.md#brand-energy)

## Color palette

Cool sky whites and grays with blue undertones. Sky blue accent (`#2B7EC2`) against cool neutral base.
Light mode first; dark mode deferred. Full palette in branding doc.

> Source: [docs/L0-branding.md](docs/L0-branding.md#color-palette-light-mode-primary)

## Typography

Geist Sans (variable weight) for UI text. Geist Mono for code/metadata.
Hierarchy via weight only — no size variation. No custom display font.

> Source: [docs/L0-branding.md](docs/L0-branding.md#typography)

## Layout

Three-column: sidebar, conversation list, message detail.
Selection flows left-to-right. Resizable panels with localStorage persistence.

> Source: [docs/L0-ui.md](docs/L0-ui.md#navigation-model)

## Non-goals (v1)

Offline mutation queue, drag-and-drop message management, custom themes, print, mobile-first layout.

> Source: [docs/L0-ui.md](docs/L0-ui.md#what-we-dont-build-for-v1)
